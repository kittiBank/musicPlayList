# Music Playlist Back-End

NestJS Hello World Project

## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Test the application

Once the application is running, open your browser and navigate to:
```
http://localhost:3000
```

You should see "Hello World!" message.
